Mob programming

    Informally: Mobbing
    What does it mean? It is a software development approach where the whole team works on the same thing, at the same time, in the same space, and at the same computer.(Link 1)
    During the initial meeting you can familiarize each other with the project. Start by taking a look at some code, database tables, documents, and a few other details of the project. At some point start to try out a few ideas, writing tests, changing code, and discussing how to proceed. (Link 2)
    Remote Mob Programming combines two ways of working: Mob Programming and working as a distributed team. With Remote Mob Programming, collaboration will be closely in the same virtual space.(Link 3)
